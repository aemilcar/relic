Jeeves
======

Cloud server management tool

Jeeves requirements
-------------------

Jeeves depends on the following software:

- Python 2.7 
- Django >1.4 (`sudo pip install Django`)
- RabbitMQ

And the following Python packages:

	boto
	pika
	PyMySQL
